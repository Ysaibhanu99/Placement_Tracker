# 🚀 194-Day Placement Prep Tracker

A beautifully designed, fully interactive single-page placement preparation tracker built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

> Target: **Full Stack Developer / AI-ML Engineer** | **8–12 LPA** | Placements: **January 2027**

---

## ✨ Features

### 📅 Live Day Tracking
- Automatically calculates the current day based on real date
- Dynamically highlights today's row, marks past days as completed
- Shows real calendar dates alongside day numbers

### ✅ Interactive Checklists
- Every task has a clickable checkbox
- Progress saved to `localStorage` — persists across browser sessions
- Checked tasks get a strikethrough effect

### 🎯 Today's Mission Panel
- Dedicated spotlight card showing only today's tasks
- SVG completion ring showing live progress (e.g., 3/5 = 60%)
- No scrolling needed to see what to do right now

### ⏱️ Live Countdown
- Real-time countdown to placement date (January 2027)
- Updates every second — days, hours, minutes, seconds

### 🎉 Confetti Celebration
- Complete ALL of today's tasks and get a colorful confetti explosion
- Fires only once per day — a small daily reward

### 🌙☀️ Light & Dark Theme
- Toggle between dark (default) and light theme
- Theme choice saved to `localStorage`
- Smooth transitions between themes

### 🔥 Streak Flame
- Animated fire emoji appears next to your streak when it hits 5+ days
- Visual motivation to keep the streak alive

### ⌨️ Keyboard Shortcut
- Press **`T`** anywhere to instantly scroll to today's row

### 📊 Progress Tracking
- Animated progress bar with glowing tip and shimmer effect
- Phase markers (Foundation → Core Skills → Interview Ready → Final Sprint)
- Weekly mini-stats showing tasks completed this week

### 🎨 Unique Design
- **Animated mesh gradient** background with drifting color orbs
- **Floating particles** on canvas
- **Holographic card borders** with rotating conic-gradient
- **3D tilt** effect on stat cards
- **Space Grotesk + JetBrains Mono** typography
- **Noise texture** overlay for analog premium feel
- **Scroll-triggered** entrance animations
